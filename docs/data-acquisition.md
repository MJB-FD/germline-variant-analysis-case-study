# Data Acquisition Plan

## Project
Germline Variant Analysis Case Study

## Dataset Source
1000 Genomes Project

## Selected Strategy
Chromosome 22 subset from a single human sample

## Rationale
Chromosome 22 is the smallest autosome in the human genome, making it suitable for low-resource variant calling while still representing real germline variation.

## Data Type
Paired-end Illumina short-read sequencing (FASTQ)

## Expected Pipeline Input
- FASTQ files (raw reads)
- Reference genome (GRCh38 or GRCh37)

## Next Step
Download a small subset of sequencing reads for one individual and prepare for quality control analysis.

No data has been downloaded yet. This document defines the acquisition strategy for reproducibility.
