# Dataset Description

## Project
Germline Variant Analysis Case Study


## Data source
1000 Genomes Project


## Sample Type 
Human germline DNA sequencing data


## Objective
Identify germline variants relative to human reference genome.


## Rationale
The 1000 Genomes Project is a widely used public resource for human genetic variation studies and provides suitable data for demonstrating variant-calling workflow.


## Notes
A small subset of publicly available data will be used so that the workflow can run on a low-end linux laptop.
