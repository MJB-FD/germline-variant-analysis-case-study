# Analysis Plan

## Dataset 

Public germline variant data from 1000 Genome Project.

## Scientific Question

Which germline variants are present in the selected individual, and what biological insights can be derived from them?

## Planned Analyses

### 1. VCF Exploration 

- Inspect VCF structure
- Count variants
- Identify chromosomes represented

### 2. Variant Filtering

- SNPs
- Indels 
- Quality-based filtering

### 3. Functional Annotation

- Gene association
- Coding consequences
- Predicted impact

### 4. Biological Interpretation

- High-impact variants
- Population context
- Potential biological relevance

## Deliverables

- Reproducible workflow
- Annotated variant dataset
- Summary statistics
- Final report
