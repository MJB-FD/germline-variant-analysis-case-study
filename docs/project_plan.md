# Germline Variant Analysis Case Study

## Scientific Question

Which germline variants can be idetified from a public human sequencing sample and what biological insights can be obtained from those variants?

## Objectives 

1. Obtain public sequencing data.
2. Perform sequencing quality control.
3. Align reads to a reference genome.
4. Call germline variants
5. Filter low-confidence variants.
6. Annotate variants.
7. Interpret biologically relevant findings.

## Dataset 

Public human sequencing data from 1000 Genome Project.

## Reproducibility

All commands, software versions, and outputs will be documented in Github.
