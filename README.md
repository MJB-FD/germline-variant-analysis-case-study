# Germline Variant Analysis Case Study

## Objective

Build a reproducible germline variant analysis workflow using public sequencing data.

## Workflow

1. Downlaod public sequencing data
2. Perform quality control
3. Align reads to a reference genome
4. Call germline variants
5. Filter variants
6. Annotate variants
7. Interpret biological findings

## Repository Structure

data/
results/
scripts/
docs/
env/

## Reproducibility 

All analysis are performed on Linux using documented commands and version-controlled code.
